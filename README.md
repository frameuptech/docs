Bienvenue dans la documentation de FrameUp

## Manuel d'utilisateur de FrameUp

FrameUp est une plateforme qui vous permet de gérer facilement du contenu à distance pour un ou plusieurs écrans connectés. Vous téléchargez des médias (vidéos, images, fichiers audio, documents) sur le portail, configurez des mises en page et des plannings, et poussez à distance votre contenu vers tous les écrans de votre compte. Pour ce faire, **FrameUp** nécessite un lecteur connecté à chaque écran que vous souhaitez gérer. Les lecteurs peuvent être achetés via **FrameUp**.
